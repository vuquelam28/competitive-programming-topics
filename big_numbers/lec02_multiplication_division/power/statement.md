# Lũy Thừa (Big Number)

Cho trước hai số nguyên dương $a$ và $b$.

***Yêu cầu:*** Hãy tính $a^b?$

## Input

- Một dòng duy nhất chứa hai số nguyên dương $a$ và $b$.

## Constraints

- $1 \le a \le 12$.
- $1 \le b \le 12000$.

## Output

- In ra trên một dòng kết quả của bài toán.

## Sample Input 1

```
2 3
```

## Sample Output 1

```
8
```

## Sample Input 2

```
5 4
```

## Sample Output 2

```
625
```
