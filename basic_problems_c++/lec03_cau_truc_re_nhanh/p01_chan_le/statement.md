# Chẵn Lẻ

Nhập vào số nguyên $n$ từ bàn phím.

***Yêu cầu:*** Kiểm tra xem $n$ là số chẵn hay số lẻ?

## Input

- Dòng duy nhất chứa số nguyên dương $n$.

## Constraints

- $0 \le n < 10^9$.

## Output

- Dòng duy nhất in ra `ODD` nếu $N$ là số lẻ hoặc in ra `EVEN` nếu $n$ là số chẵn.

## Sample Input 1

```
3
```

## Sample Output 1

```
ODD
```

## Sample Input 2

```
4
```

## Sample Output 2

```
EVEN
```
