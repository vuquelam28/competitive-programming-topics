# Chữ số đầu tiên

Cho số nguyên $n$.

***Yêu cầu:*** Bạn hãy viết chương trình **đệ quy** cho biết chữ số đầu tiên của $n$.

## Input

- Chứa duy nhất số nguyên $n$.

## Constraints

- $1 \le n \le 10^{18}$

## Output

- Chữ số đầu tiên của $n$.

## Sample input 1

```
103
```

## Sample output 1

```
1
```
