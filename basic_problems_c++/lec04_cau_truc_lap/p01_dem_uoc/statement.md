# Đếm Ước

Cho trước số nguyên dương $N$.

***Yêu cầu:*** Hãy đếm số lượng ước nguyên dương của $N?$

## Input

- Chứa duy nhất số nguyên dương $N$.

## Constraints

- $1 \le N \le 10^6$.

## Output

- Số nguyên duy nhất là số lượng ước nguyên dương của $N$.

## Sample Input 1

```
10
```

## Sample Output 1

```
4
```

## Explanation 1

Các ước nguyên dương của $10$ là: $1, 2, 5, 10$.
