# Tam Giác Cân Rỗng

Nhập vào số nguyên dương $H$ từ bàn phím

***Yêu cầu:*** In ra màn hình một hình tam giác cân rỗng gồm các ký tự $*$ và có độ cao $H$ (như hình vẽ).

## Input

- Dòng duy nhất chứa số nguyên dương $H$.

## Constraints

- $0 < H \leq 10$.

## Output

- Tam giác cân rỗng có độ cao $H$.

# Sample input 1

```
4
```

## Sample output 1

```
   *
  * *
 *   *
*******
```
