# Biểu Thức

Cho một số nguyên dương $N$.

***Yêu cầu:*** Hãy tính biểu thức sau:
$$S(N) = 1 + 1.2 + 1.2.3 + \cdots + 1.2.3...N$$

## Input

- Chứa duy nhất số nguyên dương $N$.

## Constraints

- $1 \le N \le 100$.

## Output

- Số nguyên duy nhất là kết quả biểu thức $S(N)$.

## Sample Input 1

```
3
```

## Sample Output 1

```
9
```

## Explanation 1

$S(3) = 1 + 1.2 + 1.2.3 = 9$.
