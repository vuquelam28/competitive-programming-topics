# Chữ Số Lớn Nhất

Viết chương trình nhập vào số nguyên dương $N$ và tìm chữ số lớn nhất của $N$.

## Input

- Dòng duy nhất chứa số nguyên dương $N$.

## Constraints

- $0 < N < 10^9$.

## Output

- Số nguyên dương $K$ là chữ số lớn nhất của $N$.

# Sample input 1

```
231
```

## Sample output 1

```
3
```
