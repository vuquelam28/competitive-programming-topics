# Mũ 2 Liên Tục

Cho số $N$.

***Yêu cầu:*** Tìm số $k$ lớn nhất sao cho $2^k \le N?$

## Input

- Dòng đầu tiên chứa số $N$ duy nhất.

## Constraints

- $1 \le N \le 10^{18}$.

## Output

- Một số $k$ lớn nhất sao cho $2^k \le N$.

## Sample input 1

```
6
```

## Sample output 1

```
2
```


## Sample input 2

```
1000000000000000000
```

## Sample output 2

```
59
```
