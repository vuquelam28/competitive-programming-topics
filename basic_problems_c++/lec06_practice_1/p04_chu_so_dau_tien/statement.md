# Chữ Số Đầu Tiên

Viết chương trình nhập vào số nguyên dương $n$ từ bàn phím. 

***Yêu cầu:*** Hãy tìm chữ số đầu tiên của $n?$

## Input

- Dòng duy nhất chứa số nguyên dương $n$.

## Constraints

- $1 \le n \le 10^{18}$.

## Output

- Số nguyên dương $K$ là chữ số đầu tiên của $N$.

# Sample Input 1

```
231
```

## Sample Output 1

```
2
```
