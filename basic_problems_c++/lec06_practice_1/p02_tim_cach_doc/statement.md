# Tìm Cách Đọc

Viết chương trình nhập một số nguyên dương $N$ có hai chữ số.

***Yêu cầu:*** Hãy in ra cách đọc của số nguyên này?

## Input

- Dòng duy nhất chứa số nguyên dương $N$. 

## Constraints

- $10 \leq N \leq 99$.

## Output

- Dòng duy nhất in ra cách đọc của số nguyên dương $N$.

# Sample input 1

```
28
```

## Sample output 1

```
Hai muoi tam
```
