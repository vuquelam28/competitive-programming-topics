# Kiểm Tra Quý

Viết chương trình nhập vào tháng $M$ trong năm $2022$.

***Yêu cầu:*** Cho biết tháng $M$ thuộc quý mấy trong năm?

## Input

- Dòng duy nhất chứa số nguyên dương $M$.

## Constraints

- $0 < M < 13$.

## Output

- Dòng duy nhất in ra các số từ $1$ đến $4$ là quý mà tháng $M$ thuộc.

# Sample input 1

```
9
```

## Sample output 1

```
3
```
